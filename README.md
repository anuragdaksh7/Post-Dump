# Post-Dump

### A platform where you can share any content ( image ) anonymously.

### No user guidelines.

## Installation guide
- git clone https://github.com/anuragdaksh7/Post-Dump.git
- #### for server
  - cd server
  - npm install
  - npm run dev
- #### for client
  - cd client
  - npm install
  - npm run dev
